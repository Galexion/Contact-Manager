The way to find Satellaview servers!

Commands
--------
| Command | Discription |
|--|--|
| !help | Brings Up The Help Menu |
| !server <Region / Server Name> | Shows Details about Server / Shows Servers avalible in your Region. |
| !invite | Invite this bot to your Server! |
